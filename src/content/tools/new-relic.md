## What is New Relic?

New Relic is a widely used software-as-a-service APM tool. It gathers telemetry data (metrics, logs, traces) from web applications and databases, organizing them into visual performance dashboards.

## Why use it in the API Lifecycle?

* **Transaction Tracing**: Identifies slow API endpoints and provides a breakdown of database query execution and external service wait times.
* **Error Analytics**: Groups API exceptions, logs errors, and alerts teams before outages affect users.
* **Custom Dashboards**: Build charts mapping client geographies, active API key usages, and throughput rates.

## Best Practices

* Set up New Relic alerts using baseline thresholds to automatically detect anomalies in API request durations.
