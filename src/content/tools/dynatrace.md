## What is Dynatrace?

Dynatrace is an enterprise observability platform. It uses an agent-based model (OneAgent) to automatically discover services, map topologies, and trace API transactions across large infrastructures.

## Why use it in the API Lifecycle?

* **Automatic Service Mapping**: Instantly discovers and maps call topologies between microservices without manual configuration.
* **Root Cause Analysis**: Uses AI to automatically pinpoint the root cause of API outages or performance degradations.
* **Full-stack Context**: Connects API transaction traces directly to system CPU, memory, virtual machines, and container metrics.

## Best Practices

* Leverage Dynatrace when monitoring large Kubernetes clusters or multi-cloud enterprise application deployments.
