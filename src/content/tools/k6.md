## What is k6?

k6 is a developer-centric, open-source load testing tool developed by Grafana Labs. It is written in Go and executes performance scenarios scripted in clean, ES6 JavaScript.

## Why use it in the API Lifecycle?

* **As-Code Configuration**: Script load tests in JavaScript, keeping performance validation configurations versionable inside Git.
* **Resource Efficient**: Go runtime engine enables k6 to generate massive virtual user traffic (thousands of parallel requests) using minimal system resources.
* **CI/CD Thresholds**: Set performance budgets (e.g., p95 latency must be under 200ms) to fail deployment pipelines if performance degrades.

## Best Practices

* Run smoke tests with k6 in CI/CD on every release, and run heavy spike/stress tests on staging environments before key launches.
