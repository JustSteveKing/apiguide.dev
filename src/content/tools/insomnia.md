## What is Insomnia?

Insomnia is a streamlined, lightweight desktop client for debugging REST, SOAP, GraphQL, and gRPC endpoints. It focuses on offering a fast interface for local developers.

## Why use it in the API Lifecycle?

* **GraphQL Native**: Includes auto-complete schema capabilities for GraphQL APIs out of the box.
* **Environment Chains**: Chain request variables (e.g. extracting a token from a POST login response and passing it automatically as a Bearer token in subsequent requests).
* **Fast Startup**: Lighter and faster than resource-heavy alternatives.

## Best Practices

* Export your Insomnia workspaces into JSON configurations to share with your engineering team.
