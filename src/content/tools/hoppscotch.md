## What is Hoppscotch?

Hoppscotch is an open-source API client that runs in the browser (or as a self-hosted/desktop app), offering a lightweight alternative to heavier GUI clients like Postman or Insomnia. It supports REST, GraphQL, WebSocket, Server-Sent Events, and MQTT requests from a single interface, with no required account or local install to get started.

## Why use it in the API Lifecycle?

* **Zero-Install Testing**: Runs entirely in the browser, letting developers fire off requests and inspect responses without installing a desktop client or creating an account.
* **Self-Hostable**: Teams with data residency or compliance requirements can self-host the full application instead of relying on the hosted SaaS version.
* **Multi-Protocol Support**: Handles REST, GraphQL, WebSockets, and other protocols in one client, reducing the need to switch tools when a project mixes API styles.
