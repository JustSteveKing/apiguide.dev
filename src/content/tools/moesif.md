## What is Moesif?

Moesif is an API analytics platform that tracks user behavior and API performance. Unlike generic server monitors, Moesif ties API requests directly to specific user profiles and accounts.

## Why use it in the API Lifecycle?

* **User Debugging**: Track the exact sequence of API calls that a failing customer made, helping customer support resolve integration issues.
* **Usage-Based Billing**: Aggregate API counts to meter billing and integrate directly with payment processors like Stripe.
* **API Product Metrics**: Understand which features are most popular, which clients are hitting rate limits, and where integration drop-offs occur.

## Best Practices

* Use Moesif's alert rules to notify account managers when a customer's API integration suddenly drops in volume, indicating a broken client implementation.
