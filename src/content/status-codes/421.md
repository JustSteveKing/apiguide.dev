## What does HTTP status code 421 mean?

The 421 Misdirected Request status code indicates that the client directed its request to a server that is unable to generate a response (e.g. DNS or connection reuse issues).

## When to use it in APIs

Useful in HTTP/2 settings when a client attempts to reuse a connection for a domain name that the server is not configured to support.

## How it compares to other codes

* **404 Not Found**: Implies the path doesn't exist on this host, whereas 421 implies the request reached the wrong host entirely.
